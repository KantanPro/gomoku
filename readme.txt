WordPressサイトに五目並べゲームを組み込むプラグインです。15×15のボードで遊べる本格的な五目並べゲームを提供します。

== Description ==

Gomoku Gameは、WordPressサイトに五目並べゲームを簡単に組み込むことができるプラグインです。

**主な機能：**

* **15×15のゲームボード** - 標準的な五目並べのサイズ
* **AI対戦モード** - 初級・中級・上級の3段階の難易度
* **キャラクターモード** - 石、絵文字、ファンタジーなど6種類の表示
* **ダークテーマ対応** - システム設定に応じた自動テーマ切り替え
* **ゲーム履歴** - ログインユーザーのみ記録・表示
* **ゲーム管理機能** - リセット、一手戻し、履歴表示
* **スコア記録** - ゲーム結果をデータベースに保存
* **レスポンシブデザイン** - モバイルデバイスでも快適に遊べます
* **ショートコード対応** - `[gomoku]`で簡単にページに表示
* **管理画面** - ボードサイズ、AI難易度、テーマなどの設定が可能
* **GitHub連携** - 自動更新チェックと通知機能

**ゲームルール：**
黒と白が交互に石を置き、縦・横・斜めのいずれかの方向に5つ並べた方が勝利です。

**使用方法：**
ページや投稿に `[gomoku]` ショートコードを挿入するだけで、ゲームが表示されます。

**カスタマイズ：**
* `[gomoku board_size="19"]` - ボードサイズを変更
* `[gomoku ai_level="hard"]` - AI難易度を設定（easy/medium/hard）
* `[gomoku dark_theme="dark"]` - テーマを強制設定（auto/light/dark）
* `[gomoku character_mode="emoji"]` - キャラクターモードを設定

== Installation ==

1. プラグインファイルを `/wp-content/plugins/gomoku/` ディレクトリにアップロードします
2. WordPressの管理画面でプラグインを有効化します
3. ページや投稿に `[gomoku]` ショートコードを挿入します
4. 必要に応じて管理画面の「Gomoku Game」設定でカスタマイズします

== Frequently Asked Questions ==

= ゲームはどのように表示されますか？ =

`[gomoku]` ショートコードをページや投稿に挿入することで、ゲームが表示されます。

= ボードサイズは変更できますか？ =

はい、管理画面の設定で15×15、19×19、13×13から選択できます。また、ショートコードのパラメータでも指定可能です。

= AI対戦は可能ですか？ =

はい、AI対戦モードを有効にすることで、コンピュータと対戦できます。

= スコアは保存されますか？ =

設定で有効にすると、ゲーム結果がデータベースに保存され、統計情報として表示されます。

= モバイルデバイスでも遊べますか？ =

はい、レスポンシブデザインに対応しており、スマートフォンやタブレットでも快適に遊べます。
